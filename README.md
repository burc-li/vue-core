# vue2-core

### 介绍
本仓库用于学习Vue2设计思想和理念，手写部分核心源码

### 安装教程
1.  cd 1.vue-render (2.vue-dep ...)
2.  npm install
3.  npm run dev
4.  使用浏览器打开 dist 目录下的 html 测试文件 （vscode 推荐使用 Live Server 插件）

### 章节介绍
#### vue-render
主要介绍 响应式原理 和 模版编译

#### vue-dep
主要介绍 依赖收集

#### vue-set
主要介绍 vm.$set 原理

#### vue-mixin
主要介绍 Vue.mixin 原理
